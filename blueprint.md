# 🚀 WORDBOOM: AI 맞춤형 단어 정복 서비스

## 1. 프로젝트 개요
사용자가 단어장 사진을 업로드하면, **AI(Gemini)**가 이미지 속 텍스트를 정교하게 분석하여 [영단어 : 뜻] 쌍으로 추출합니다. 이를 바탕으로 고품질의 **PDF 시험지/정답지**를 자동 생성하고, 학습을 돕는 **인터랙티브 단어 게임**을 제공합니다.

## 2. 디자인 컨셉: "Vibrant Harmony"
기존의 심심한 디자인에서 탈피하여, 학습의 즐거움을 줄 수 있는 유니크하고 컬러풀한 디자인을 적용합니다.
- **Glassmorphism**: 반투명 배경과 블러 효과를 활용한 현대적이고 세련된 UI.
- **Dynamic Gradients**: Indigo, Violet, Soft Pink 등 활기찬 컬러를 조화롭게 배치.
- **Micro-interactions**: 버튼 및 카드 요소에 은은한 'Glow' 효과와 부드러운 애니메이션 적용.
- **Navigation Bar**: 상단 네비게이션을 통해 업로드, 시험지 생성, 게임 섹션으로의 직관적인 이동 지원.

## 3. 핵심 기능

### 3.1. [AI 추출] 스마트 이미지 분석
- **Gemini Vision API**: 단순 OCR을 넘어 문맥을 파악하여 오타를 교정하고 정확한 단어 쌍을 추출합니다.
- **인터랙티브 미리보기**: 추출된 단어를 즉시 수정하거나 추가할 수 있는 리스트 UI 제공.

### 3.2. [자동 생성] 맞춤형 시험지 & 정답지
- **디자인 템플릿**: 학교 시험지 스타일부터 모던한 학습지 스타일까지 세련된 PDF 레이아웃 제공.
- **자동 페어링**: 단어 시험지와 정답지가 한 번에 생성되어 학습 효율 극대화.

### 3.3. [플레이] 사용자 단어 기반 '악당 퇴치' 게임
- **시각적 강화**: 게임 캔버스에 네온 효과와 파티클 애니메이션을 추가하여 몰입감 증대.
- **난이도 조절**: 단어의 길이나 학습 상태에 따른 가변적 게임 속도 적용.

## 4. 기술 스택
- **Frontend**: HTML5, CSS3 (Modern Features: OKLCH, Grid/Flexbox), JavaScript (ES Modules)
- **AI Integration**: Google Gemini API (Vision)
- **Libraries**:
    - `jsPDF` & `jspdf-autotable`: PDF 생성
    - `Canvas API`: 게임 구현

## 5. 업데이트 계획
1. **네비게이션 및 레이아웃 재구현**: 상단 네비게이션 바 추가 및 반응형 그리드 시스템 적용.
2. **AI 로직 고도화**: Gemini API 연동을 위한 구조 설계 (이미지 업로드 -> AI 처리 -> 데이터 파싱).
3. **디자인 리뉴얼**: 새로운 컬러 팔레트와 글래스모피즘 스타일 적용.
4. **사용자 경험 개선**: 로딩 애니메이션, 성공/오류 메시지 등 피드백 강화.
